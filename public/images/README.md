# Team Member Images

To add Dikshant Gangawat's profile image:

1. Save the uploaded image as `dikshant-gangawat.jpg` in this directory (`public/images/`)
2. Make sure the file name is exactly: `dikshant-gangawat.jpg`
3. The image should be a square or portrait photo for best results

5. Supported formats: JPG, JPEG, PNG, WebP
